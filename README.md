# News-Websites-Crawling-And-Searching
This is a web project about crawling and searching news websites by node.js.

## 项目目录说明
APP：第一次项目
APP2：第二次项目(拓展功能)
crawlers：爬虫代码文件夹  
sample：复现示例代码文件夹  
static：前端静态文件文件夹  
templates：前端网页文件夹
sql：数据库配置文件夹
